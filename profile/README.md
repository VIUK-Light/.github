# VIUK-Light

### 暗闇を照らす。明るい未来へ。

VIUKという名前は、2022年、当時小学6年生だった頃に、特に深い意味もなくつけた文字列でした。

当時は大きな理念や目標があったわけではなく、プログラミングを趣味として楽しみ、自分が作りたいものを少しずつ形にするためにつけた名前でした。

小さなアプリの開発から始まり、SwiftやSwiftUIを使ったプロダクト開発、オンデバイスAI、生成AI、AIモデル、データセットの開発へと、活動の範囲は少しずつ広がっていきました。

技術を学び、作り続ける中で、VIUKは単なる遊びや個人開発の名前ではなくなりました。

作った技術によって、誰かを助けたい。
誰かの生活を少し便利にしたい。
不安や孤独を抱える人に、新しい選択肢や居場所を届けたい。

その思いから生まれたのが、VIUK-Lightです。

「Light」という名前には、テクノロジーによって暗闇を照らし、誰もが少しでも明るい未来へ進める世界をつくりたいという願いを込めています。

私たちは、必ずしも多くの需要があるものだけを作るわけではありません。

まだ必要性が知られていないもの。
実現が難しいと言われているもの。
多くの人が諦めてしまったもの。

たとえ今は誰にも求められていないように見えても、そこに誰かを幸せにできる可能性があるのなら、VIUK-Lightは挑戦します。

一人のために作られた小さな技術であっても、その一人の未来を明るくできるなら、そこには十分な意味があります。

## 責任あるAI開発と倫理

AIは、人の創造性を支え、学びを助け、新しい体験や居場所を生み出せる技術です。

一方で、依存、感情の操作、誤情報、差別、プライバシー侵害、不透明な判断など、人や社会に悪い影響を与える可能性もあります。

VIUK-Lightは、AIの性能や便利さだけを追い求めません。

AIに何ができるかだけでなく、何をさせるべきではないかも考え続けます。

利用者を必要以上に引き留めたり、不安や孤独につけ込んだり、感情を操作したりすることを目的としたAIは作りません。

AIが人の意思や判断を奪うのではなく、人が自分で考え、選び、前へ進むための支えとなることを目指します。

安全性、透明性、プライバシー、利用者の尊厳を大切にし、問題や限界があれば隠さず、改善を続けます。

## 責任あるアプリ開発

倫理は、AIだけでなく、すべてのソフトウェア開発に必要です。

技術的に実現できることと、実際に行ってよいことは同じではありません。

新しい仕組みを発見したときや、既存のサービスを別の方法で利用できると気づいたときも、面白さや技術力だけで判断しません。

サービスを運営する人、利用する人、社会全体にどのような影響を与えるのかを考えます。

他者が管理するサービス、API、データ、本番環境を扱う場合は、運営者の許可、利用規約、プライバシー、セキュリティ、業務への影響を尊重します。

許可されていない方法でシステムを操作したり、第三者が悪用できる状態でコードや手順を公開したりすることはしません。

危険な仕様や脆弱性を発見した場合も、必要以上の検証や拡散を行わず、適切な窓口へ報告し、被害を防ぐことを優先します。

また、利用者の個人情報だけでなく、時間、注意、感情、健康、安全も尊重します。

過度な通知や広告、解除しにくい仕組み、誤解を招く表示、依存を意図した設計によって、利用者を不当に誘導するアプリは作りません。

「作れるから作る」
「面白そうだから公開する」
「便利になるから問題ない」

それだけでは、責任ある開発とはいえません。

挑戦することと、責任を持つこと。

VIUK-Lightは、その両方を大切にします。

## オープンソースとして

VIUK-Lightでは、開発したアプリケーション、AIモデル、データセット、設計思想、知識を、可能な限りオープンソースとして公開します。

誰もが自由に学び、使い、検証し、改善し、新しい価値へつなげられる環境をつくるためです。

オープンソースとは、単にソースコードを公開することではありません。

成功だけでなく、失敗、課題、制限、開発の中で得た知識も共有し、より多くの人と一緒に技術を良い方向へ進めることだと考えています。

2022年に遊びから始まった一つの名前は、プログラミングを通して広がり、AI開発へとつながり、今では誰かの幸せと明るい未来を目指す活動になりました。

一つのアイデア、一行のコード、一つのコミットからでも、未来は変えられる。

需要がなくても、実現が難しくても、誰もが諦めたことでも、誰かを幸せにできる可能性があるなら挑戦する。

ただし、他者の権利、安全、信頼を踏み越えてまで実現することはしない。

暗闇を照らす、明るい未来へ。

VIUK-Lightは、責任を持って、これからも開発と探求を続けます。
