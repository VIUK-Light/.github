# VIUK-Light

### 暗闇を照らす。明るい未来へ。

VIUKという名前は、2022年に遊びの中で生まれました。

当時は大きな目標があったわけではなく、プログラミングを趣味として楽しみ、自分が作りたいものを少しずつ形にしていました。

小さなアプリの開発から始まり、SwiftやSwiftUIを使ったプロダクト開発、オンデバイスAI、生成AI、AIモデルやデータセットの開発へと、活動の範囲は少しずつ広がっていきました。

技術を学び、作り続ける中で、VIUKは単なる個人開発の名前ではなくなりました。

現在のVIUK-Lightが目指しているのは、テクノロジーによって、誰かの生活や未来を少しでも明るくすることです。

AIは、ただ便利な機能を提供するためだけのものではありません。人の創造性を支え、学びを助け、新しい体験や居場所を生み出し、人と人、人と技術をつなぐ可能性を持っています。

一方で、AIの便利さだけを追い求めるのではなく、プライバシー、安全性、透明性、そして利用する人自身の意思を尊重することも重要だと考えています。

VIUK-Lightでは、Swift、SwiftUI、オンデバイスAIを中心に、AIを日常の中で安心して利用できる、新しいプロダクト体験を探求しています。

私たちが目指しているのは、人をAIに依存させることでも、AIにすべてを任せることでもありません。

人とAIがそれぞれの役割を持ち、互いを補いながら、より自然につながる未来をつくることです。

また、開発したアプリケーション、AIモデル、データセット、知識は、可能な限りオープンソースとして公開していきます。

誰もが自由に学び、使い、改善し、新しい価値を生み出せる環境をつくるためです。

2022年に遊びから始まった一つの名前が、プログラミングを通して広がり、AI開発へとつながり、今では誰かを幸せにするための活動になりました。

一つの小さなアイデアや、一つのコミットであっても、誰かの未来を照らす光になるかもしれません。

誰もがテクノロジーの恩恵を受けられる、明るく、開かれた未来へ。

VIUK-Lightは、これからも開発と探求を続けていきます。
